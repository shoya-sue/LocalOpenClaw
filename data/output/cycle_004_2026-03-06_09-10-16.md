To compute the sum of the first $ n $ natural numbers efficiently, we can use the well-known **arithmetic series formula**:

$$
S = \frac{n(n + 1)}{2}
$$

This formula is derived from the observation that the sum of the first $ n $ natural numbers forms an arithmetic series where the first term is 1, the last term is $ n $, and the number of terms is $ n $. The sum can be computed by averaging the first and last terms and multiplying by the number of terms:

$$
S = \frac{1 + n}{2} \times n = \frac{n(n + 1)}{2}
$$

---

### Example Calculation

- For $ n = 5 $:  
  $$
  S = \frac{5 \times 6}{2} = 15
  $$
  (Sum: $ 1 + 2 + 3 + 4 + 5 = 15 $)

- For $ n = 10 $:  
  $$
  S = \frac{10 \times 11}{2} = 55
  $$
  (Sum: $ 1 + 2 + \dots + 10 = 55 $)

---

### Implementation in Python

You can compute this efficiently in Python using integer division to avoid floating-point precision issues:

```python
def sum_first_n_natural_numbers(n):
    return n * (n + 1) // 2
```

This function works for positive integers $ n $, and returns the correct sum of the first $ n $ natural numbers.

---

### Summary

- Formula: $ S = \frac{n(n + 1)}{2} $
- Efficient implementation: $ \text{int}(n(n + 1) // 2) $
- Example: $ \text{sum_first_n\_natural\_numbers}(5) = 15 $

This approach ensures optimal performance even for large values of $ n $, with constant time complexity $ O(1) $.
